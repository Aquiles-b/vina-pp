<A fazer> -i Opção de inserir mais membros no archive. Se um membro já existe deve
ser substituído e colocado inseridos respeitando a ordem da linha de comando,
ao final do archive;

<A fazer> -a É igual -i porém só substitui se for mais recente.

<A fazer> -m Move o membro para trás do target existente passado. Deve ocorrer
na seção de dados(?).

<A fazer> -x Extração dos membros indicados. Caso nenhum seja indicado extrair 
todos.

<A fazer> -r Remove membros indicados.

<A fazer> -c Lista o conteúdo do archive em ordem com todas as propriedade do 
membro e sua ordem no arquivo (igual comando tvf do tar).

<A fazer> -h Help para os comandos.

OBS:
    Caminhos sempre relativos.
    Deve existir uma área diretório(cabeçalho) com todas as informações dos 
    membros 
    O conteúdo dos membros devem ser manipulados em blocos de 1024 bytes.

Propriedades dos membros:
    nome (sem espaços), 
    UID (user ID),
    permissões,
    tamanho,
    data de modificação,
    ordem no arquivo (Ordem de inserção, pode ser alterada pela opção -m)
    localização.
